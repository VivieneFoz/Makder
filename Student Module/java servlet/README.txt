1. Download the project.
2. Open your ecclipse ide
3. Click file
4. Open file from project file system 
5. Choose "login" folder to open the project.
6. Left click in the login
7. Choose "Run as"
8. Run on a server
9. Choose Tomcat v8.0 Server at localhost and click Finish
10. you can access the web page through ecclipse or browser by connecting to the local host provided by Tomcat.
